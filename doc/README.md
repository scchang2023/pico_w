
# pico-w

- Raspberry Pi Pico板載26個GPIO腳位、2個SPI、2個I2C、2個UART、3個12位ADC以及16個可控PWM，內建LED是GPIO25，支持配置引腳功能，方便使用者彈性開發與應用。
